# Mason Scraper - Working Instructions

## Non-Negotiable Rules

1. **Think First**: Read codebase, understand the problem, write plan to `tasks/todo.md`
2. **Task Tracking**: Maintain checklist in `tasks/todo.md`, check off items as completed
3. **Check In**: Verify plan with user before starting implementation
4. **Progress Updates**: Mark tasks complete as you go
5. **High-Level Updates**: Give brief explanation of changes made at each step
6. **Simplicity**: Make every change as simple as possible, impact minimum code
7. **Review Section**: Add summary to `tasks/todo.md` when complete
8. **No Laziness**: Find root causes, no temporary fixes, senior developer standards
9. **Minimal Impact**: Changes should only touch necessary code, avoid introducing bugs

## Project Overview

Web scraper to extract all products from masonstores.com including:
- Product details (name, price, SKU, description, specs)
- Product images
- Export to JSON and CSV

## Key Files

- `mason_scraper.py` - Main scraper script
- `tasks/todo.md` - Task tracking
- `output/` - Generated data and images (gitignored)
